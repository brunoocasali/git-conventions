Git Convention
==============

Todos nós, desenvolvedores, trabalhamos com versionamento. E grande parte de nós optamos por trabalhar com Git, por sua popularidade e facilidade de aprendizagem.

Porém, como todo bom programador descobre cedo que é preciso padronizar seu workflow. Seja no modo em que escreve seu código, seja na arquitetura do projeto, seja no método de trabalho. 

E trabalhar com Git muitas vezes significa trabalhar em equipe. E ele facilita muito para fazer isto com mais facilidade.
